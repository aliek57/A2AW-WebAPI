# Atividade 2 de API e Web Services - Cadastro de Usuários

Desenvolva uma Web API para o gerenciamento de dados de usuários. A Web API deve permitir:
* Cadastro de usuários, contendo os campos: nome, e-mail e idade.
* Listagem dos registros previamente cadastrados.

A implementação da Web API deve ser realizada em JAX-RS e Spring Boot. Além disso, desenvolva uma interface web utilizando HTML e JavaScript para integração com a Web API.
A interface deve iniciar em uma página principal, que apresente o sistema e ofereça acesso
às páginas de cadastro e listagem de usuários. Não é necessário o uso de banco de dados;
os dados podem ser armazenados em memória.
